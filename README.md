# Nano Bird

Flappy Bird am Arduino Nano auf einem OLED Display, wobei man entweder mit knöpfen direkt ober über Roboremo am Handy mit bluetooth spielen kann.

## Hardware

- Arduino Nano
- Led LED-Streifen
- OLED Display
- 2 Buttons mit 10k Widerständen
- HC-05 Modul
- Android Handy mit Roboremo
- Kabel, Steckbrett, ...
